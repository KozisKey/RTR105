Linux commands
====================

Basic
---------------------

### whoami
   > Print username
### pwd
   > print working directory    
### ls
   > list folders and files
### who
   > who is using device
### man
   > manual for commands
### uname
   > operating system
### echo $0
   >show interpreter
### clear
   > clear termina
### ls -l
   > list folders
### ls -a
   > list all 
### cd
   > go to home directory
### cd ..**
   > go up one directory
### cd /
   > go to root directory
### history
   > command history
### ls -l *.txt
   > list documents ending with .txt
### cat *.txt
   > read and show  data in file
### cat *.txt  > *.txt
   > read, show data in file and export to another file
### toch 
   > change file
### cp 
   > coppy file
### rm 
   > remove file
### mv 
   > move file
